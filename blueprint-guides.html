Blueprint & Guides

New to the ALS Blueprint System? This guide walks you through every step — from downloading the base blueprint to getting your character running in Unreal Engine 5.

About the blueprint

The blueprint and guides we provide are designed to show how our products work in Unreal Engine. Our blueprint is based on the Advanced Locomotion System V4 by LongmireLocomotion, available free on the Unreal Engine Marketplace. We do not hold rights or ownership of that blueprint.

We provide the child blueprint for the animations — it contains the specific animation sequences and adjustments. To use this set you must first obtain the main ALS blueprint which includes the fundamental character movement logic. Our child blueprint enhances and customises those base functionalities.

Installation steps

Video walkthrough: How to install ALS and setup RPG Blueprint

1
Get ALS V4 and create a new ALS project
Download Advanced Locomotion System V4 from FAB and create a new project using the ALS template.
2
Migrate ALS_GameMode_SP to your project
Open the ALS project. Migrate ALS_GameMode_SP to your project's Content folder.
3
Hard-import the RPG_BasicMove_BP folder
Import our RPG_BasicMove_BP folder into your project's Content folder. Your Content folder should now contain these 3 folders:
📁 AdvancedLocomotionV4
📁 RPG_BasicMove
📁 RPG_BasicMove_BP
4
Import the Input_ALS key settings
In your project, go to: Edit > Project Settings > Engine Input > Import… (top right) and import the Input_ALS file we provide.
5
Add 3 new enumerators to ALS_OverlayState
Go to the ALS_OverlayState Enumerator and add 3 new entries:
+ Normal_A
+ Normal_B
+ Normal_C
6
Compile and check the RPG blueprint
Open BP_RPG_FemaleUE5_CH and compile. Check that the Switch on ALS_OverlayState node is properly connected to the 3 new enumerators (Normal_A, Normal_B, Normal_C).
7
Set GameMode in World Details — and you're done!
In your level's World Details, set:
GameMode Override → ALS_GameMode_SP
Default Pawn Class → BP_RPG_FemaleUE5_CH (or UE4 version)

Enjoy the experience! 🎮

Tip & Trick
Known Issue — Camera Jiggle

Problem: The camera jiggles while the character is falling near a platform or ledge.

Solution: Add camera sockets on the head skeleton of the character:

• FP_Camera
• TP_CameraTrace_R
• TP_CameraTrace_L
